# Проект: Сервис To-Do

## Описание
Это одностраничное приложение для работы со списком дел.  
Пользователь может просматривать задачи, добавлять новые, удалять, копировать и редактировать их.  
Все изменения сохраняются в `localStorage`, поэтому список дел не пропадает после перезагрузки страницы.

## Функциональность
- отображение стартового списка задач при загрузке страницы;
- добавление новой задачи через форму;
- удаление задачи;
- копирование задачи;
- редактирование текста задачи;
- сохранение всех изменений в локальном хранилище браузера.

## Технологии
- HTML
- CSS
- JavaScript
- БЭМ
- `localStorage`

## Структура JavaScript
В проекте реализованы основные функции:

- `loadTasks()` — загружает задачи из `localStorage` или возвращает стартовый массив;
- `saveTasks(tasks)` — сохраняет массив задач в `localStorage`;
- `getTasksFromDOM()` — собирает текущие задачи со страницы;
- `updateStorage()` — обновляет данные в `localStorage`;
- `createItem(item)` — создаёт элемент задачи на основе шаблона;
- `renderTasks(taskList)` — выводит список задач на страницу.

## Как запустить проект
1. Открыть папку проекта в редакторе кода.
2. Запустить файл `index.html` в браузере.
